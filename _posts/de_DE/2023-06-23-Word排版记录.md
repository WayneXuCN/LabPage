## Word-Makros

### Farbe für Querverweise

Dieses Makro formatiert automatisch Querverweise in Word sowie Endnote/Zotero-Querverweise. Die Standardfarbe ist Blau.

```basic
Sub CitingColor()
    For i = 1 To ActiveDocument.Fields.Count 'Durchläuft alle Felder im Dokument
        If Left(ActiveDocument.Fields(i).Code, 4) = " REF" Or Left(ActiveDocument.Fields(i).Code, 14) = " ADDIN EN.CITE" Or Left(ActiveDocument.Fields(i).Code, 31) = " ADDIN ZOTERO_ITEM CSL_CITATION" Then 'Die Feldcodes für Word-Querverweise beginnen mit " REF" (beachte das Leerzeichen), für EndNote mit " ADDIN EN.CITE" und für Zotero mit " ADDIN ZOTERO_ITEM CSL_CITATION". Weitere Typen können nach Bedarf hinzugefügt werden.
        ActiveDocument.Fields(i).Select 'Markiert die oben genannten Felder
        Selection.Font.Color = RGB(31, 77, 160) 'Setzt die Schriftfarbe
        End If
    Next
End Sub
```

### Einheitliche Skalierung für Mathtype

Dieses Makro behebt das Problem uneinheitlicher Höhen bei Mathtype-Formeln nach Aktualisierungen, indem es alle Formeln einheitlich skaliert.

```basic
Sub EqMathtype_100()
'
' EqMathtype_100 Makro
''
Dim i As Integer
Dim total As Integer
total = ActiveDocument.InlineShapes.Count
i = 0
For Each s In ActiveDocument.InlineShapes
i = i + 1
On Error Resume Next
Application.StatusBar = "Fortschritt: " & i & " von " & total
If s.Type = msoAutoShape Then
If s.OLEFormat.ClassType = "Equation.DSMT4" Then
With s
.ScaleHeight = 100
.ScaleWidth = 100
End With
End If
End If
Next
End Sub
```
